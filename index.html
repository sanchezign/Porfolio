Más sobre mí

Sobre mí y mi trabajo

¿Qué tecnologías usás?
Trabajo con React, Next.js y Astro en el frontend, con Tailwind CSS para estilos. En el backend uso Node.js, Express, MongoDB y MySQL. Para infraestructura manejo Docker y Git, testing con Mocha, Jest y Supertest, y TypeScript cuando el proyecto lo requiere.
Mi experiencia
Desde 2023 trabajé con dos clientes reales — Golden Pets y Pura Vida Huertas (rebranding de Pura Vida) — entregando sus sitios completos de principio a fin. Cada entrega fue un producto real en producción, con responsabilidades concretas y soluciones a medida.
Mi formación
Completé la carrera Full Stack Developer en Coderhouse con 6 cursos y más de 100 horas: Desarrollo Web, Javascript, React JS y tres niveles de Backend culminando en Testing y Escalabilidad con Docker. Todo lo apliqué en proyectos reales.
¿Cómo trabajás en equipo?
Me adapto bien a equipos y también trabajo de forma autónoma — algo que aprendí gestionando proyectos para clientes reales. Uso Scrum y Kanban según el contexto, Git y GitHub para el control de versiones, y Trello o Jira para organizar el trabajo.
¿Disponible para freelance?
Sí, estoy disponible para proyectos freelance y también abierto a oportunidades de trabajo en relación de dependencia. Si tenés un proyecto en mente, ¡escribime!
Metodologías ágiles
He trabajado con Scrum y Kanban. En Scrum participé de sprints con revisiones periódicas y pull requests antes de mergear a producción. Con Kanban usé tableros para gestionar el flujo de tareas y participé de dailies para reportar avances.
